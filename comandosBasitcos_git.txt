Commit -> E uma forma de enviar as modificação que estamos realiazando no projeto. Ele vem 
atrelado a um comentário.

git status -> é usado para verificar as alterações realizadas, usando mais esse comando 
git add -A ele adicina no repositório

Para subir o aquivo para o github, usamos esse comando git commit -m, ele realiza somente as
modificações que foram alteradas não o projeto todo.
	.Usando o comandao git add -A, ele confirmará a alteração feita no documento.

Usando o git log conseguiremos ver todos os commits tudo que foi feito

Como Bob mencionou, você pode alterar diretórios com cd:
